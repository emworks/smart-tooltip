# Smart tooltip

Application to display step-by-step guide using tooltips.

# Setup

## Prerequisites

- Docker
- Node.js 12.8.0

## Build with Docker

```
docker build -t smart-tooltip .
docker run -p 9000:9000 smart-tooltip
```

Demo application will be started at http://localhost:9000.

## Build with NPM

```
npm install
npm run build
npm start
```

Demo application will be started at http://localhost:9000.
